Eine .NET Developer Group oder auch INETA User Group ist ein Zusammenschluss von Softwareentwicklern, die mit der Microsoft Software-Plattform .NET arbeiten.

Die INETA ist eine unabhängige Non-Profit-Organisation, die alle Gruppen unterstützt, die sich für die .NET-Plattform interessieren. Die INETA wird von Usergroup-Vertretern geführt, die alle Usergroup-Vertreter aus ihrer Mitte dazu gewählt haben. Die INETA wird finanziell von Microsoft und anderen Sponsoren gefördert.

Die .NET DevGroup Göttingen/Kassel ist aus der VOCA-Usergroup Göttingen/Kassel hervorgegangen und besteht seit dem 4. Dezember 1997.
